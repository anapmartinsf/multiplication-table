# Tabuada
Este projeto se trata de uma aplicação simples para gerar tabuadas de multiplicação dinamicamente, com base nos números fornecidos pelo usuário.

## Tecnologias utilizadas:
 - HTML;
 - CSS;
 - JavaScript;
